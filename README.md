enlargeable jquery plugin
===========

This plugin can make an element enlargeable :).

Rather than words, this is an <a href="http://jsfiddle.net/popallo/fwVwJ/" target="_blank">example link</a>.

```javascript

    tooltip:true,
    targetWidth:'958px',
```
